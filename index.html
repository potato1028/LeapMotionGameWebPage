LeapMotion 센서를 활용한 실시간 손동작 인식 기반 재활게임

게임 스크린샷

문제 정의

디지털 기기의 과도한 사용으로 인해 손의 소근육 발달이 지연되고 있다는 우려가 커지고 있습니다. 손과 손목의 활발한 사용은 일상생활뿐만 아니라 유아기 발달과 치매 예방에도 중요한 역할을 합니다. 기존 손 재활 도구들은 높은 가격과 낮은 맞춤성으로 인해 접근성이 제한되어 있습니다. 이를 해결하고자 컴퓨터 비전 AI 센서인 립모션을 활용하여 손의 움직임을 실시간으로 분석하고 손 기능 개선과 발달을 돕는 게임 기반의 손 재활 및 발달 플랫폼을 개발하였습니다.

전체 기술 스택

Unity C# C++ C Python Git Notion React.js TypeScript JavaScript Chart.js Spring Boot MySQL AWS

아키텍처 / 설계

문제 해결을 위해 LeapMotion과 Unity 연동을 핵심 전략으로 채택했습니다. MySQL과 AWS RDS를 활용해 데이터베이스를 구축하고, React.js, TypeScript, JavaScript, Chart.js, CSS, Spring Boot, AWS를 활용하여 웹 페이지를 구축하여 게임 데이터를 연동하였습니다.

핵심 기능

1. 싱글톤 패턴을 이용한 최적화

싱글톤 패턴을 활용하여 게임의 데이터 관리를 효율화 하였습니다.

2. LeapMotion SDK

LeapMotion SDK를 사용하여 유니티 내에서 사용자의 손 동작 데이터를 받아와 게임에 적용시켜 원활한 게임 플레이가 가능하도록 만들었습니다.

나의 역할 및 기여

  • LeapMotion SDK와 유니티의 연동
  • 손 동작 데이터를 기반으로 한 게임 진행 방식 고안 및 제작
  • 전체적인 게임 플레이 방식 확립 (효과음, 사운드 적용, 게임 데이터 관리 등)

결과 및 성과

LeapMotion을 활용하여 실시간으로 사용자의 손동작을 게임 내 데이터로 활용 가능
(손바닥 기울기, 주먹 쥐기, 손가락 펴기, 검지 포인팅, 박수, 손 뒤집기 등)
